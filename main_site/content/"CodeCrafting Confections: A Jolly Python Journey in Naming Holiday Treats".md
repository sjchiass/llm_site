**The Code**

Here be the Python code that'll generate ye some merry Christmas treat names:
```python
import random

# List o' words to use in our Christmas treat names
adjectives = ["Jolly", "Sparkly", "Sweet", "Merry"]
nouns = ["Cookies", "Candies", "Gingerbread", "Peppermint"]

def generate_treat_name():
    adjective = random.choice(adjectives)
    noun = random.choice(nouns)
    return f"{adjective} {noun}"

# Generate 10 treat names and print 'em out!
for _ in range(10):
    print(generate_treat_name())
```
**How it Works**

This code uses Python's `random` module to select words from two lists â€“ `adjectives` and `nouns`. We then define a function called `generate_treat_name()` that picks one word from each list and combines 'em into a tasty treat name. Finally, we use a loop to generate 10 names and print 'em out!
